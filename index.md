## 🛠️ Cara Instalasi & Aktivasi Pro

### 1. Cara Install
1. Unduh file `AeroLauncher.exe` terbaru melalui tombol download di atas.
2. Pindahkan file tersebut ke folder aman (misalnya `C:\Program Files\AeroLauncher`).
3. Jalankan aplikasinya! 

> ⚠️ **Catatan Windows SmartScreen:** Karena aplikasi ini baru dirilis dan belum memiliki sertifikat berbayar, Windows mungkin akan memunculkan peringatan *"Unknown Publisher"*. Jangan khawatir! Cukup klik **"More Info"** lalu pilih **"Run Anyway"**. Aplikasi ini 100% aman dan bebas malware.

### 2. Cara Aktivasi Fitur Pro
1. Selesaikan pembayaran di halaman resmi **Lemon Squeezy** melalui tombol *Upgrade ke Pro*.
2. Periksa email Anda untuk menyalin **License Key** yang dikirimkan otomatis.
3. Buka AeroLauncher di PC Anda, klik ikon **Settings (Gerigi)** -> **Activate Pro**.
4. Tempelkan kunci lisensi Anda dan klik **Verify**. Selamat menikmati fitur premium!
